## GitHub 常用搜索条件

| 索引 | 条件        | 命令                | 说明      |
| ---- | ----------- | ------------------- | --------- |
| 1    | 搜索名字    | in:name xxx         |           |
| 2    | 搜索描述    | in:description xxx  |           |
| 3    | 搜索 readme | in:readme xxx       |           |
| 4    | 按 stars    | stars:>2000         |           |
| 5    | 按 fork     | fork:>3000          |           |
| 6    | 按仓库大小  | size:>=5000         | 单位为 kb |
| 7    | 按更新时间  | pushed:>2020-01-01  |           |
| 8    | 按语言      | language:javascript |           |
| 9    | 按作者名    | user:xxx            |           |

搜索的方式可以组合，更多高级搜索：https://github.com/search/advanced
